## Overview

**Greendex** is a DLSU (De La Salle University) GPA tracking single-page application. It lets students calculate per-term GPA, cumulative GPA (CGPA), project future CGPA scenarios, calculate weighted grades, and export/print grade cards as images. Data persists via Firebase Firestore for authenticated users, with sessionStorage fallback for anonymous users.

## Data Flow

1. **Authentication:** `App.tsx` → Firebase Auth (`onAuthStateChanged`) → sets `user` state
2. **Data Loading:** Component mount → checks `user` → if logged in: Firestore via `firestore.ts`; if anonymous: sessionStorage via `storage.ts`
3. **Data Saving:** Course change → auto-save to Firestore (authenticated) or sessionStorage (anonymous)
4. **Migration:** On first login, GPACalculator migrates anonymous sessionStorage data → Firestore
5. **GPA Calculation:** Client-side only — `useMemo` computes from courses array
6. **Print/Export:** `PrintGradesModal` → renders `PrintPreview` → `dom-to-image-more` captures to PNG → download or browser print

### Firestore Data Model
```
users/{userId}/
  ├── terms/{termNumber}        → { courses: Course[], isFlowchartExempt: boolean }
  ├── settings/projections      → { targetCGPA: number, totalUnits: number }
  ├── settings/cgpa             → { creditedUnits: number }
  └── gradeCalculator/subjects  → { subjects: Subject[] }
```
